8. Programa

Este programa determina si un valor y las operaciones internas que se ingresen cumplen con las características de un grupo o semigrupo.

El programa contiene varias funciones que verifican los axiomas característicos, por lo que, el programa solo solicita un 'click' en verificar para ejecutar las funciones.

Se ingresarán 2 tipos de datos: "Conjunto" y "Operaciones".

El programa verificará si cumplen para ser 'grupo' o 'semigrupo'.



Para poder definir un 'Conjunto', este se debe ingresar con comas y sin espacios; EJEMPLO: 1,2,3,4

Para poder definir una 'Operación' se deben ingresar dos elementos del 'Conjunto' con el simbolo de + , - , * o %; EJEMPLO: 1+2; 3-1; 4*1; 3%1

Si se quiere ingresar más de una operación la forma de ingresarlo es la siguiente: (1+2)+(3+4).



Para formar un GRUPO puedes intentar la siguiente combinación [Conjunto (1,2,3,4,5); Operación (1+2) o (1*2) o (3-1)]

Para formar un SEMIGRUPO puedes intentar la siguiente combinación [Conjunto (0,1,2,3,4,5); Operación (0*2) o (0%3)]
